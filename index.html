Exemplos D3

A proposta desse laboratório é criar visualizações com D3, para isso utilizamos dados de pessoas tentando ser aletórias e avaliando o quão aleatórias elas foram. Os dados foram coletados usando esse form: o quão aleatórios somos?, os dados utilizados seguem esse formato.

Para construir nossas visualizações tivemos que fazer um processamento nos dados. Os scrips estão disponíveis no repositório no github.
